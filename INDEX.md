## 风格挑选建议

**根据内容类型：**
- 数据/报告 → 01 或 10
- 个人作品/介绍 → 04 或 05
- 产品 Landing → 02 或 06
- 长文/采访 → 09
- 生活方式 → 07
- 个性表达 → 03 或 08

**根据色彩偏好：**
- 想要深色高级 → 01, 02, 03
- 想要明亮干净 → 05, 06, 10
- 想要温柔 → 07
- 想要有个性 → 03, 08
- 想要复古 → 09
- 想要叙事感 → 04, 09

## 使用方式

```
读 templates/{编号-名称}/template.html  # 完整 HTML
读 templates/{编号-名称}/tokens.md       # 设计变量速查
```

替换 `[[XXX]]` 占位符为用户实际内容，不改 CSS。
